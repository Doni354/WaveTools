# 🔨鸣潮工具箱
🔨鸣潮工具箱是一个用于修改鸣潮画质和帧率的工具箱

⚠️仅支持 x64、arm64 架构的 Windows 10 1809 及以上版本的系统
# 功能
### 当前版本的工具箱包括以下功能：
🔓帧率解锁

⚙️画质修改

# 正在计划的功能
🗃️抽卡分析

🔑账号切换

# 安装方法
打开Powershell

输入irm wavetools.jamsg.cn/get | iex

回车后将自动安装并启动WaveTools

**如出现闪退状况，尝试以管理员运行Powershell后运行**set-executionpolicy remotesigned**后再试**

# 注意事项
本工具箱仅适用于PC版鸣潮。
